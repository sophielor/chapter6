CHAPTER 6

HTML: Anchor and Hyperlinks



hreflang
- The "hreflang" attribute is used to specify the language of the linked resource in HTML, and it must be used together with the "href" attribute. HTML5 uses language values from BCP 47, while HTML4 uses language values from RFC 1766.

rel
- The "rel" attribute specifies the relationship between the current HTML document and the linked document. For HTML5, the values for this attribute must be defined in the HTML specification or registered in the Microformats wiki.

target
- The target attribute in HTML is used to specify where to open the link when clicked, and it can take several values such as "_blank", "_self", "_parent", "_top", and "framename" (deprecated). However, it is not recommended to force this behavior as it can violate the user's control over a website.

title
- The "title" attribute is used to provide additional information about a link, and the information is usually displayed as a tooltip when the cursor hovers over the link. It can be used on almost all HTML tags, not just links.

download
- The "download" attribute specifies that the target of a hyperlink should be downloaded when clicked, and the attribute's value is the name of the downloaded file. The browser will automatically detect the file extension and add it to the file name if it's not specified. If the attribute's value is omitted, the original filename will be used.

The anchor tag, also known as < a > tag, can be used to link separate web pages, as well as different sections within the same document. It can be used in table of contents or to launch external applications.
